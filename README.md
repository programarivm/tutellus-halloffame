# Tutellus Hall of Fame

Builds a randomish hall of fame of Tutellus instructors.

---

### Set up the Environment

Create an `.env` file:

    $ cp .env.example .env

Build a hall of fame as an HTML file:

    $ php cli/load.php > hall.html
